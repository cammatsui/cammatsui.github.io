RateOurCourses.com

RateOurCourses.com is a student-run course and professor review website at Amherst College. Since taking over backend development of the site, I have refactored all of the site's backend PHP code, added professor-rating and confirmation email features, and restored the course and review databases.

Catan Board Generator

A website to generate boards for the Settlers of Catan board game. Allows for choice in resource distribution, desert/port placement, and number distribution.

ModularEpi

ModularEpi is a framework written in Python to easily create and simulate compartmental epidemiological models. Features include metapopulation models and R0 calculation with the Next Generation Matrix method.

Conway's Game of Life

A JavaScript application to run the mathemetician John Horton Conway's cellular automaton the Game of Life.
